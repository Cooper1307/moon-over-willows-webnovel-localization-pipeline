# 角色专属档案: 老祖宗


#### 来自 第16章

| 维度 | 详情 |
|------|------|
| **身份与外貌** | 一百二十岁，朱府辈分最高的长辈。白发、满脸褶皱但有白皙的光泽，目光炯炯有神 |
| **性格特征** | 洞察世事，超脱凡俗。并未因朱父的权威而盲从，而是给朱淑真一个证明自己才华的机会 |
| **英文语调** | **Wise, serene, authoritative yet detached** |
