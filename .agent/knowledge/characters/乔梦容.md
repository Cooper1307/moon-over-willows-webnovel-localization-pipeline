# 角色专属档案: 乔梦容


#### 来自 第6章

| 维度 | 详情 |
|------|------|
| **身份背景** | 8岁时被赵朴路边捡回，由于与乔贵妃同姓，备受宠爱并封为郡主 |
| **外貌性格** | 长相俊俏，端庄典雅，温柔如水，典型的大家闺秀 |
| **情感关系** | 与赵朴青梅竹马。对赵朴的感情深厚且单纯，甚至有些盲目崇拜 |
| **英文语调** | **Meek, dramatic, overly formal** (如SOP所述)，对哥哥的称呼带有亲昵与敬畏 |


#### 来自 第7章

| 维度 | 详情 |
|------|------|
| **身世背景** | 6岁被遗弃路边，被赵朴马车救回。视赵朴为救赎与唯一的依靠 |
| **情感倾向** | 即使赵朴不醒或失忆，也愿意为其守节。比起赵朴的权势，更在意他的平安 |
| **处事方式** | 懂礼数、识大体，主动通过贿赂内侍来保护赵朴，展现出贤内助的潜质 |


#### 来自 第31章

| 维度 | 详情 |
|------|------|
| **身世背景** | 原名木梦容，被赵朴捡回。赵朴为纪念生母乔贵妃，并取《诗经》“迁于乔木”之意，让她改姓乔 |
| **情感表现** | 看到哥哥练武感到陌生，但很快自我安慰是“上天垂怜”。主动拥抱哥哥，发现他心跳加快并未推开自己，感到一丝欣慰，对哥哥充满依恋 |


#### 来自 第32章
- **本章状态**：对申简辰（哥哥）充满爱意和渴望，回忆起以往两人相处的点滴，努力迎合他的喜好，想要成为他的女人。
- **英文语调**：继续保持 **Meek, dramatic, overly formal**（柔弱、戏剧化、过于正式），对话中带有明显的依恋和讨好色彩（clingy/affectionate）。


#### 来自 第33章
- **本章状态**：继续默默依恋男主（靠在男主肩头感到幸福）。心思细腻，察觉到哥哥心中“另有其人”，暗自伤神，同时对乔文仙口中的“已婚女子”感到好奇。
- **英文语调**：保持 **Soft, overly formal, inwardly melancholic (柔和、过于正式、内心忧郁)**。

---


#### 来自 第37章
- **本章状态**：对男主（哥哥）的依恋进一步加深，甚至把男主“剽窃”的咏梅词误认为是写给自己的情诗，陷入自我感动的回忆中。面对男主的疏离感到失落，但只要男主说一句“一家人”，就又会感到温馨。
- **英文语调**：Delusional in love, overly sensitive, desperately clinging (沉溺于幻想的爱、过度敏感、拼命攀附)。
