# 角色专属档案: 朱晞颜


#### 来自 第5章

| 维度 | 详情 |
|------|------|
| **身份** | 临安知府 |
| **性格/爱好** | 典型文人官员，喜爱诗词歌赋，将女儿培养成才，在文人圈中有一定影响力 |
| **家庭地位** | 掌握女儿婚配权，林觉认为他很难同意和离 |


#### 来自 第11章

| 维度 | 详情 |
|------|------|
| **性格特征** | 极其严肃，重清誉。被下人畏惧，与女儿关系冷淡如上下级 |
| **家庭地位** | 朱府的绝对权威，所有重大决定（包括家人同桌吃饭）都需要他点头 |
| **英文语调** | **Strict, distant, authoritative** |


#### 来自 第13章

| 维度 | 详情 |
|------|------|
| **性格特征** | 极其冷酷无情，重男尊女卑、重声誉胜过女儿生死。认为被家暴是女儿不圆房的错 |
| **态度** | 对女儿重生的说法无动于衷，认为是"乡野之人胡乱编造"。坚决反对和离 |
| **英文语调** | **Cold, unyielding, deeply patriarchal and unsympathetic** |


#### 来自 第14章

| 维度 | 详情 |
