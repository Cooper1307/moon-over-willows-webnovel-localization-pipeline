# 角色专属档案: 朱淑真


#### 来自 第3章

| 维度 | 详情 |
|------|------|
| **身份** | 临安知府千金，宋代女词人，现为林觉之妻 |
| **年龄** | 约20岁（比现代年轻了10岁） |
| **身体状态** | 额头受轻伤（被林觉撞晕），身体素质由原本娇弱变为现代武术灵魂掌控 |
| **性格表现** | 面对家暴果断反击，言语犀利（"老娘"上线），拥有现代人的法律意识 |
| **语言风格** | 现代吐槽与古代硬气结合。如："老娘既然嫁给他，就是他的正室" |
| **翻译要点** | 突出由于"换魂"带来的巨大反差，尤其是武力值压制时的爽快感 |


#### 来自 第4章

| 维度 | 详情 |
|------|------|
| **外貌细节** | 古典精致的美人，长发漆黑如墨且浓密（可接洗发水广告级别） |
| **心态转化** | 正式决定接纳朱淑真的身份生活下去，但保留现代人的灵魂内核（如对铜镜的不解、饮食的豪放） |
| **清白状态** | 极其庆幸朱淑真此前以死相逼未与林觉圆房，保住了身体的清白 |
| **英文语调** | 自信、豪放、带点幽默感（"跟小姐混，肯定不会被人欺负"） |


#### 来自 第5章

| 维度 | 详情 |
|------|------|
| **文学才华** | 通过那首《眼儿媚·迟迟春日弄轻柔》展现其原本娇俏、多愁善感的女性形象 |
| **社交圈** | 婚后依然经常与文人好友、才子郊游玩乐，表现出对包办婚姻的无声反抗 |

#### 来自 第8章

| 维度 | 详情 |
|------|------|
| **法律意识** | 敏锐察觉到"和离"胜于"被休"，并开始考虑如何说服视名誉如生命的父母 |
| **性格魅力** | 甚至让林觉觉得"更有趣了"。以前的原主只写诗、冷漠、不说话；现在的她爽朗、爱嘚啵（话多）、有主见 |
| **情感细节** | 潜意识里挂念申简辰（喊他的名字），但嘴上依然嫌弃他是"损友" |
| **待人处事** | 恩威并施。解开林觉的束缚，但立下三条核心底线（不准进屋、不准带妓女回家、必须给钱） |


#### 来自 第9章

| 维度 | 详情 |
|------|------|
| **书写技能** | 继承了现代千里寻七八年的书法功底，字迹与原主有七八分相似 |
| **家庭关系** | 与父母关系紧张，因不满婚事半年未回娘家。目前嫂嫂刚生产（或待产） |
| **文学共鸣** | 通过读《浣溪沙·春夜》，真正体会到了原主那份"恨情和梦更无聊"的深沉孤寂 |
| **现代背景** | 母亲早逝，父亲传武后出家。独自撑起千里拳庄，养成独立坚韧的性格 |


#### 来自 第10章

| 维度 | 详情 |
|------|------|
| **情感认同** | 第一次真正感受到古代家庭的温暖，因向夫人的关怀而产生归属感（蹲下靠在母亲膝头） |
| **社交焦虑** | 对"江南女词人"这一头衔感到压力。自认是诗词门外汉，通过回忆原主作品来"应试" |
| **英文语调** | 在母亲面前表现出一种"依恋"而"坦诚"的口吻，同时不失现代人的果敢 |
