# 角色专属档案: 白风


#### 来自 第37章
- **本章状态**：在医铺为了要回五两银子又哭又闹，强词夺理，展现出市井小民的狡黠与倔强。遇到男主后，立刻见风使舵，高兴地跟着男主回王府。
- **英文语调**：Loud, obstinate, opportunistic (大嗓门、固执、见风使舵)。
