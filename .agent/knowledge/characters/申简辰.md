# 角色专属档案: 申简辰


#### 来自 第2章

| 维度 | 详情 |
|------|------|
| **全名** | 申简辰 |
| **职业** | 大公司CEO + 申拳第360代传人 |
| **外貌** | 西装革履时深邃帅气，休闲装时随意不失帅气 |
| **座驾** | 劳斯莱斯 |
| **性格特征** | 1. 对千里寻直球追求（略笨拙） 2. 霸道但有柔情 3. 战斗中温柔克制 4. 浪漫（引用古诗形容"约会"） |
| **语言风格** | 句式简练，有时调侃，偶尔深情 |
| **英文语调** | Composed, authoritative, slightly possessive, romantically oblivious |
| **代表台词** | "今天一大早起来，看到你成为全网焦点，赶过来瞧瞧。" |
| **代表台词** | "月上柳梢头，人约黄昏后。" |
| **翻译要点** | 他把"决斗"理解为"约会"——这种反差萌是角色核心 |


#### 来自 第6章

| 维度 | 详情 |
|------|------|
| **原主状态** | 体弱多病，不受官家重视，性格温和但平庸 |
| **申简辰进入后** | 身体奇迹般康复，脉象强劲。举止不再病态，眼神犀利，说话声如洪钟 |
| **自我定位** | 快速适应王爷身份。虽不懂历史，但很享受"饭来张口、衣来伸手"的CEO级别待遇 |
| **情感核心** | 心系千里寻。对乔梦容展现出一种陌生的疏离感与礼貌的抗拒 |


#### 来自 第7章

| 维度 | 详情 |
|------|------|
| **真实年龄** | 18岁（穿越后比现代年轻了12岁，这是他最满意的惊喜） |
| **性格转型** | 从原本的"含情脉脉、病态温情"转变为"阳刚、有力、甚至有些冷淡"。具备CEO的处事能力，能得体应对内侍 |
| **社会地位** | 彻底确认是"不被重视的王爷"，父皇从未探望，仅通过内侍传话 |
| **行为逻辑** | 不愿通过贿赂上位，但为了宽梦容的心选择妥协。决定"认真做个不被重视的仪王"，享受自由 |


#### 来自 第31章

| 维度 | 详情 |
|------|------|
| **情感核心** | 心里依然只装着千里寻，对乔梦容的亲密接触（拥抱）感到不自在和抗拒，认为乔梦容是赵朴的菜，不是自己的菜 |
| **生活习惯** | 习惯了现代公司里一群男弟子跟着的生活，对王府里全是一群妙龄丫鬟伺候感到很不自在，想找近身侍卫和武林好汉 |
| **心理活动** | 得知自己名下有大量良田铺子、无需为王府开销发愁后，感到非常开心和放心（不爱妹妹爱田产） |


#### 来自 第32章
- **本章状态**：开始适应古代生活，展现出前世现代人的习惯（如按照现代军训模式训练侍卫、对古代常识的匮乏，以为脚店是足浴），同时对寻找女主（千里寻）充满执念。
- **英文语调**：在面对侍卫时表现出 **Authoritative, composed**（威严、从容），在乔梦容面前则有些 **Awkward/Detached**（尴尬、疏离），内心独白带有现代人的幽默与自嘲。


#### 来自 第33章
- **本章状态**：展现了强大的武力值和决断力（果断收下根骨好的白风）。对女主的思念愈发强烈，一听到“南边”（临安）就迫不及待想出发。
- **英文语调**：在教训白风时表现出 **Strict but benevolent (严厉但仁慈)**；在面对乔文仙时，展现出穿越者在新世界找到亲友的 **Warm and relieved (温暖、欣慰)**；内心独白依然带有现代人的吐槽（如“只要不叫他舅舅，怎么称呼都无所谓”）。


#### 来自 第34章
- **本章状态**：得知自己穿越成了宋徽宗的儿子，且周围全是历史上有名的奸臣，从一开始的“发憷”变为“轻松与不屑”。他带着现代人的上帝视角审视这段历史。
- **英文语调**：内心独白充满 **Sarcastic, dismissive, modern (讽刺、不屑、现代感)**。在面对文渊时，由于差点说漏嘴（提到《水浒》），表现出 **Quick-witted but slightly clumsy recovery (机智但有些笨拙的圆谎)**。


#### 来自 第35章
- **本章状态**：在朝堂上公然“瞎编”起死回生的经历来暗讽朝政；在宴会上展现豪放的酒量（连干四杯）；面对即将到来的“品画”考验，从最初的毫无头绪到准备靠“拍马屁”过关。
- **英文语调**：对官家是 **Feigned respect, thinly veiled sarcasm (假装恭敬、暗藏讽刺)**；与魏夫人、乔文仙在一起时则是 **Relaxed, convivial, bold (轻松、欢乐、豪爽)**。

---


#### 来自 第36章
- **本章状态**：完美展现了“腹黑”与“圆滑”的一面。借用现代画展学到的赏画技巧和毛主席的《咏梅》词，把宋徽宗的马屁拍得天花乱坠，赢得了十个金碗。随后在蹴鞠场上，凭借申拳功底和现代足球技术，碾压了大奸臣高俅，顺利赢得了去临安的官职。
- **英文语调**：在官家面前是 **Exaggeratedly reverent, silver-tongued (极度夸张的恭敬、巧舌如簧)**；内心独白则是 **Smug, cynical, triumphant (得意、愤世嫉俗、胜利者姿态)**；在球场上展现出 **Dominant, fierce, mocking (极具统治力、凶悍、嘲弄)**。


#### 来自 第37章
- **本章状态**：对乔梦容的感情感到内疚和无奈，只能用冷处理的方式拉开距离。得知宋江已死后感到失落，但在医铺偶遇白风并成功将其收为侍卫后，又重拾心情准备南下临安。
- **英文语调**：面对乔梦容时是 **Awkward, evasive, guilt-ridden (尴尬、回避、内疚)**；面对白风时恢复了 **Authoritative, decisive (威严、果断)**。


#### 来自 第40章
- **本章状态**：抵达临安后，急于去饭庄、酒肆等“带口”的地方寻找女主。面对乔家众人的跪拜感到极不习惯，坚持平等相待。在路上对乔梦容的刻意靠近依然保持警惕和拒绝。
- **英文语调**：对待下属（白风）是 **Strict but caring (严厉但关切)**；对待亲人（乔家人）是 **Humble, modern, egalitarian (谦逊、具有现代平等观念)**；对待乔梦容依然是 **Distanced, deliberately obtuse (疏离、故意装傻)**。


#### 来自 第41章
- **本章状态**：面对乔梦容时，内心充满因无法回应其感情而产生的愧疚；面对乔文仙的单相思，展现出朋友间的仗义。最重要的是，在看到“千年神柳”时，情绪彻底失控流泪，这是他穿越以来最脆弱、最深情的一刻，随后用轻功掩饰自己的失态。
- **英文语调**：平时是 **Calm, rational, brotherly (冷静、理智、兄弟般的仗义)**；见到神柳时瞬间变得 **Overwhelmed, deeply emotional, vulnerable (情绪崩溃、深情、脆弱)**。


#### 来自 第42章
- **本章状态**：从焦急寻人（盯着街上每个人看）到悟出“顺其自然”的道理。在聚香楼遇到林觉时，展现出冷静旁观的姿态，并在关键时刻指示白风出手相助。
- **英文语调**：从 **Anxious, searching (焦急、寻觅)** 变为 **Philosophical, detached observer (富有哲理、超然的旁观者)**。


#### 来自 第43章
- **本章状态**：冷静的控场者。拦住乔文仙下跪，用“皇亲国戚”四两拨千斤地镇住林觉；在陈浩宇挑衅林觉时，又拦住乔文仙，选择“看狗咬狗”的热闹。
- **英文语调**：Authoritative, shrewd, calculating, detached (威严、精明、算计、超然)。


#### 来自 第44章
- **本章状态**：出手救了林觉，展现了高强的武力值和一定的格局。在听说书时，听到熟悉的小说情节（女娲、红香、玄英）而确信找到了千里寻的线索，激动得甚至落泪，但得知对方是个“提供话本的才子”后又陷入了失落（以为女主穿越成了男人）。
- **英文语调**：在救人时是 **Righteous, rational (充满正义感、理智)**；在听说书时是 **Deeply moved, vulnerable, then utterly confused (深受感动、脆弱，随后完全困惑)**。


#### 来自 第45章
- **本章状态**：夜晚看到神柳影子，生出浪漫与落寞交织的情怀。面对乔梦容的热情拥抱，感到沉重与尴尬；面对白风的“神助攻”（买栗子糕），更加觉得无奈。
- **英文语调**：在独处时是 **Romantic, melancholic (浪漫、落寞)**；在面对乔梦容时是 **Awkward, burdened, polite but distant (尴尬、负担重、礼貌但疏远)**。


#### 来自 第46章
- **本章状态**：对文人们伤春悲秋地品鉴别人（朱淑真）的隐私感到不屑和无聊。回忆起穿越前与千里寻在现代讨论《断肠谜》的甜蜜时光，展现出对女主深深的怀念。
- **英文语调**：在现实中是 **Bored, slightly dismissive (无聊、略带不屑)**；在回忆中则是 **Eager, deeply affectionate, nostalgic (热切、深情、怀旧)**。


#### 来自 第47章
- **本章状态**：对文人们反复剖析朱淑真伤疤的行为感到不适和反感，认为这是侵犯隐私。随后急切地向众人打听“千里寻”的下落，得知对方可能是“落魄书生”后，内心充满担忧和保护欲。
- **英文语调**：Disapproving, pragmatic (不赞同、务实)；在打听千里寻时变得 **Anxious, fiercely protective (焦急、极度护短)**。
