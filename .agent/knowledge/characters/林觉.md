# 角色专属档案: 林觉


#### 来自 第3章

| 维度 | 详情 |
|------|------|
| **身份** | 练兵官，朱淑真的丈夫 |
| **外貌** | 二十来岁，长相一般，身材魁梧，眼神不满/不屑 |
| **性格特征** | 大男子主义，家暴倾向，虚伪（称暴力为"教育"），迷恋青楼女子 |
| **战斗表现** | 空有蛮力，在真正的现代拳法面前不堪一击 |
| **语言风格** | 傲慢，使用长辈/主导者口气（"嫁出去的女儿泼出去的水"） |
| **翻译要点** | 语气需令人厌恶，展现出那种传统的、封建式的偏执 |


#### 来自 第4章

| 维度 | 详情 |
|------|------|
| **现况** | 被朱淑真五花大绑关在屋内，由于被打怕了开始尝试言语服软（叫"娘子"） |
| **心理** | 对妻子的变化感到"既熟悉又陌生"，认为以前的朱淑真幽怨酸臭且清高 |

---


#### 来自 第5章

| 维度 | 详情 |
|------|------|
| **文学素养** | 自认愚笨，对诗词毫无兴致，唯独记住了朱淑真的一首词（爱情的力量） |
| **复杂心态** | 既迷恋朱淑真的美貌，又受挫于她的冷漠；既是施暴者，内心又保留着最初的一见钟情 |
| **认知障碍** | 反复强调"你不是朱淑真"，因为无法接受柔弱妻子变身武力值爆棚的"母夜叉" |


#### 来自 第8章

| 维度 | 详情 |
|------|------|
| **心态变化** | 虽然被关了一夜，但并没有暴怒，反而因为妻子肯跟他说话而感到新奇。承认如果朱淑真早像这样有趣，他不会打她 |
| **老辣之处** | 虽然答应和离，但设下3天限期，且要求必须请到朱父。他知道朱父极重名誉，这是在反向给朱淑真出难题 |
| **男性自尊** | 虽然被打怕了，但在金钱和家庭管理上还保留着大男人的大方（准许她继续使银子） |


#### 来自 第17章

| 维度 | 详情 |
|------|------|
| **心理变化** | 被千里寻主动上药的行为迷惑，产生"她对我变好"的错觉，暴露出他对女主依然有占有欲或感情。坚决不想和离 |
| **英文语调** | **Confused, slightly hopeful, then turning defensive and cold** |


#### 来自 第21章

| 维度 | 详情 |
|------|------|
| **心理变化** | 开始对性情大变的“朱淑真”（千里寻）产生好感，甚至尝试阅读王维诗集以拉近距离。但同时坚决反对和离 |
| **行为动作** | 试图阻拦千里寻参加诗会并发生冲突，被千里寻轻松挡下。偷偷阅读《王右丞诗集》 |
| **态度** | 坦白曾因听闻别人议论朱淑真与东方尘泽等人的闲言碎语而生气动手。现在表示更喜欢有功夫、能交流的现在的妻子 |


#### 来自 第22章

| 维度 | 详情 |
|------|------|
| **行为策略** | 在长辈面前表现得能言善辩，巧妙地将两人矛盾归结于朱淑真不愿圆房、漠视自己以及与文人来往，试图借此挽回婚姻并获得长辈支持 |
| **承诺** | 当众保证只要不和离，以后不再打人，不再找商妓（前提是允许他纳妾），并允许朱淑真参加诗会 |


#### 来自 第23章

| 维度 | 详情 |
|------|------|
| **行为** | 看不懂《断肠谜》，被解谜后依然不愿放手。得知朱淑真不符合“七出”无法被休时，心中暗喜 |
| **心理** | 对朱淑真有强烈的占有欲，不愿看到她未来投入其他男人（如东方尘泽、乔文仙）的怀抱 |


#### 来自 第24章

| 维度 | 详情 |
|------|------|
| **心理活动** | 对朱淑真打出带香气的拳法感到惊艳甚至看呆。和离虽然失败了，但他却发现自己根本高兴不起来 |
| **行为** | 坚决不同意和离或休妻。面对千里寻暴露身份和武功，他半真半假地帮其圆谎（说她偷偷练习拳法），以此打消众人的疑虑，死死绑住这段婚姻 |
| **性格** | 心胸狭窄、睚眦必报。他提出和离的初衷并不是真的想离，而是想借长辈之手当众教训朱淑真 |


#### 来自 第25章

| 维度 | 详情 |
|------|------|
| **心理** | 承认不放走朱淑真从理性上是错的，但感性上舍不得。享受朱淑真作为“林觉娘子”带给他的虚荣感，尤其是能让临安才子们“看得到吃不到”，对此感到面上有光 |
| **行为** | 威胁要派士兵抓捕在府外捡拾朱淑真诗词废纸的人，但被苦荞怼回 |
