# 角色专属档案: 林觉


#### 来自 第3章

| 维度 | 详情 |
|------|------|
| **身份** | 练兵官，朱淑真的丈夫 |
| **外貌** | 二十来岁，长相一般，身材魁梧，眼神不满/不屑 |
| **性格特征** | 大男子主义，家暴倾向，虚伪（称暴力为"教育"），迷恋青楼女子 |
| **战斗表现** | 空有蛮力，在真正的现代拳法面前不堪一击 |
| **语言风格** | 傲慢，使用长辈/主导者口气（"嫁出去的女儿泼出去的水"） |
| **翻译要点** | 语气需令人厌恶，展现出那种传统的、封建式的偏执 |


#### 来自 第4章

| 维度 | 详情 |
|------|------|
| **现况** | 被朱淑真五花大绑关在屋内，由于被打怕了开始尝试言语服软（叫"娘子"） |
| **心理** | 对妻子的变化感到"既熟悉又陌生"，认为以前的朱淑真幽怨酸臭且清高 |

---


#### 来自 第5章

| 维度 | 详情 |
|------|------|
| **文学素养** | 自认愚笨，对诗词毫无兴致，唯独记住了朱淑真的一首词（爱情的力量） |
| **复杂心态** | 既迷恋朱淑真的美貌，又受挫于她的冷漠；既是施暴者，内心又保留着最初的一见钟情 |
| **认知障碍** | 反复强调"你不是朱淑真"，因为无法接受柔弱妻子变身武力值爆棚的"母夜叉" |


#### 来自 第8章

| 维度 | 详情 |
|------|------|
| **心态变化** | 虽然被关了一夜，但并没有暴怒，反而因为妻子肯跟他说话而感到新奇。承认如果朱淑真早像这样有趣，他不会打她 |
| **老辣之处** | 虽然答应和离，但设下3天限期，且要求必须请到朱父。他知道朱父极重名誉，这是在反向给朱淑真出难题 |
| **男性自尊** | 虽然被打怕了，但在金钱和家庭管理上还保留着大男人的大方（准许她继续使银子） |


#### 来自 第17章

| 维度 | 详情 |
|------|------|
| **心理变化** | 被千里寻主动上药的行为迷惑，产生"她对我变好"的错觉，暴露出他对女主依然有占有欲或感情。坚决不想和离 |
| **英文语调** | **Confused, slightly hopeful, then turning defensive and cold** |
