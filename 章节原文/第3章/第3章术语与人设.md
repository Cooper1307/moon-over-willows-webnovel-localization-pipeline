## 二、本章角色人设

### 朱淑真（女主 - 穿越后）

| 维度 | 详情 |
|------|------|
| **身份** | 临安知府千金，宋代女词人，现为林觉之妻 |
| **年龄** | 约20岁（比现代年轻了10岁） |
| **身体状态** | 额头受轻伤（被林觉撞晕），身体素质由原本娇弱变为现代武术灵魂掌控 |
| **性格表现** | 面对家暴果断反击，言语犀利（"老娘"上线），拥有现代人的法律意识 |
| **语言风格** | 现代吐槽与古代硬气结合。如："老娘既然嫁给他，就是他的正室" |
| **翻译要点** | 突出由于"换魂"带来的巨大反差，尤其是武力值压制时的爽快感 |

### 林觉（女主原配丈夫 / 本章主要对立角色）

| 维度 | 详情 |
|------|------|
| **身份** | 练兵官，朱淑真的丈夫 |
| **外貌** | 二十来岁，长相一般，身材魁梧，眼神不满/不屑 |
| **性格特征** | 大男子主义，家暴倾向，虚伪（称暴力为"教育"），迷恋青楼女子 |
| **战斗表现** | 空有蛮力，在真正的现代拳法面前不堪一击 |
| **语言风格** | 傲慢，使用长辈/主导者口气（"嫁出去的女儿泼出去的水"） |
| **翻译要点** | 语气需令人厌恶，展现出那种传统的、封建式的偏执 |

### 苦荞（配角 - 丫鬟）

| 维度 | 详情 |
|------|------|
| **特征** | 忠诚、胆怯但敢于为了小姐顶撞林觉（以朱父官衔威胁） |
| **心态转化** | 从哭哭啼啼的绝望到被小姐武力值折服的兴奋 |
| **翻译要点** | 语气应体现出护主心切，建议使用诚恳且略带哭腔到惊喜的转换 |

## 三、本章翻译挑战

1. **家暴语境的本地化**：林觉将殴打称为"教育"，以及春金巧的挑拨离间，需要准确传达那种令人气愤的封建压迫感。
2. **"老娘"的古今碰撞**：千里寻在古代环境中自称"老娘"，对周围人（林觉、春金巧）产生的震慑感和滑稽感需要通过译文体现。
3. **官职威压的翻译**：朱父作为"临安知府"对林觉"练兵官"的职业压制，需要让西方读者理解两者之间品级的差距（Prefect vs Military Training Officer）。
4. **宋代生活细节**：发髻、金钗、古装的细节描写，以及"商妓"这一特定历史词汇的精准选取。
5. **对话的戏剧张力**：院子里的决斗部分，林觉的"不屑"到"惊恐"的转折，以及千里寻干净利落的动作指令（"拿绳子来！"）。
